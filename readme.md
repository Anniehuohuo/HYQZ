# HYQZ 项目说明

本项目基于 CRMEB（ThinkPHP 6 + 前端 uni-app + 管理端 Vue）搭建，包含：

- 商城商品（含虚拟商品：网盘课件）
- 首页 AI 引流助手（对话 + 课件推荐 + 技能课超市入口）
- 技能课超市（按不同主题/能力的智能体列表与对话）

## 目录结构（核心）

- [crmeb](file:///f:/项目/HYQZ/crmeb)：后端（ThinkPHP）
- [template/uni-app](file:///f:/项目/HYQZ/template/uni-app)：用户端（H5/小程序）

## 首页 AI 助手：课件推荐规则

首页 AI 助手接口：`/api/ai/home_chat`

### 标记协议（前端解析用）

- 推荐商品：`[[RECOMMEND_PRODUCTS:ID]]`（最多 1 个）
- 不推荐商品：`[[RECOMMEND_PRODUCTS:]]`
- 无匹配时引导矩阵：`[[GO_AGENT_MATRIX]]`

前端页面会：

- 识别并移除上述标记后再展示回答正文
- 若有 `ID`，拉取商品详情并显示课件卡片
- 若出现 `[[GO_AGENT_MATRIX]]` 且未推荐课件，则展示“去技能课超市聊”的入口

### 匹配思路（简化）

- 先从可推荐商品池中按“用户输入 vs 商品名称/简介/关键词”做相关性排序
- 只把高相关 TopN 候选交给大模型选择
- 匹配度过低则强制不推荐，并引导去技能课超市继续聊

## 网盘课件（一次性发货）

后台创建虚拟商品（网盘课件）时，支持两种发货：

- 固定网盘信息：填写一段文本（可写“链接 + 提取码”）
- 一次性网盘信息：逐条录入或导入多组“链接（可选）/提取码（必填）”

用户支付成功后，系统会在订单备注/站内信中给出对应的网盘信息（链接/提取码）。
