# 慧圆智慧父母（微信小程序）需求讨论稿

> 目标：明确“慧圆智慧父母”微信小程序的信息架构、核心流程、后台配置能力与MVP范围，便于后续进入设计与开发。

## 1. 项目概述

“慧圆智慧父母”是一款面向父母人群的微信小程序，核心为可对话的智能体服务。在对话过程中，智能体可结合用户需求弹出“推荐卡片（课件）”，引导用户购买并完成交付。

## 2. 信息架构（底部导航四页）

### 2.1 首页（对话智能体）

- 默认进入主智能体对话页（类似AI客服/顾问）
- 在对话中可插入“推荐卡片（课件）”
- 目标：对话解决问题 + 推荐内容承接转化

### 2.2 技能课超市

- 多种类智能体按场景/能力提供服务（如亲子沟通、学习习惯、情绪管理等）
- 每个智能体为独立入口与独立会话
- 关键要求：支持后台配置智能体参数（例如智谱智能体ID、API Key等），便于不定时调整

### 2.3 论坛

- 用户交流与经验分享，建立圈子与内容社区
- 初期可先做MVP功能，审核能力按“上线底线”配置

### 2.4 我的（个人中心）

- 用户信息、订单与已购内容管理
- 收藏/点赞/发帖记录、设置、客服等入口

## 3. 首页核心闭环：对话 → 推荐卡片 → 购买 → 交付

### 3.1 触发机制（推荐卡片出现时机）

- 用户表达需求（问题/困扰/场景）后，智能体识别意图
- 输出正常对话回复，并附带 1–3 张推荐卡片（课件）
- 推荐频率可控（避免刷屏）

### 3.2 推荐卡片（课件）信息结构（建议最小字段）

- 标题、封面
- 1句话收益点/适用场景
- 价格、是否已购状态
- 行动按钮：去看看 / 立即购买 / 已解锁

### 3.3 购买链路（最小可用）

- 点击卡片 → 课件详情页（大纲、适用人群、截图/试阅等）
- 下单 → 微信支付 → 支付成功回跳
- 支付成功后：
  - 卡片状态更新为“已解锁”
  - “我的-已购”中可查看课件内容
  - 对话里提示用户可立即使用课件内容

### 3.4 交付形态（先从最省事开始）

- 已购后在“我的-已购”查看课件（图文/PDF/页面内容均可）
- 对话可支持引用课件要点/练习清单，增强即时价值感

## 4. 技能课超市：后台可配置能力（重点）

### 4.1 设计原则

- 小程序端不直接持有第三方模型 API Key
- 所有大模型调用由后端代理转发（前端仅调用业务接口）

### 4.2 后台配置（MVP建议字段）

- 智能体基础信息：名称、分类、简介、封面、启用状态、排序
- 供应商与模型：provider（例如智谱）、model
- 智能体标识：智谱智能体ID（如适用）
- 密钥配置：API Key（加密存储）
- 对话策略：System Prompt、温度、最大输出等

### 4.3 运行时策略（建议）

- 支持热更新（修改后立即生效）
- 支持版本化与回滚（保留上一版本，必要时快速回退）
- 记录配置变更审计日志（操作人、时间、变更差异）

## 5. 论坛：MVP与审核/风控建议

### 5.1 MVP功能建议

- 帖子列表/详情
- 发帖、评论、点赞
- 举报

### 5.2 审核与风控（上线底线）

- 发布内容走微信内容安全校验（文本/图片）
- 命中风险：拦截或进入待处理队列
- 事后治理：举报 + 后台处置（下架/删除/禁言等）
- 全量人工预审成本高，建议先不做强制人工预审流程

## 6. 数据模型（草案）

### 6.1 用户与权限

- User：openid/unionid、昵称头像、手机号绑定、状态等

### 6.2 智能体与对话

- Agent：名称、分类、provider、model、agentId、apiKey(加密)、prompt、启用、排序、版本
- Conversation：会话（user + agent）
- Message：消息（role、content、时间、关联推荐卡片等）

### 6.3 课件与交易

- Product/Courseware：课件商品（标题、内容载体、价格、上下架）
- Order：订单（金额、状态、关联用户与商品）
- Payment：支付（微信支付单号、回调结果、状态）
- Purchase/Entitlement：已购权益（用于鉴权访问课件）

### 6.4 论坛

- Post：帖子
- Comment：评论
- Like：点赞
- Report：举报
- Topic/圈子（可后置）

### 6.5 后台管理

- AdminUser、Role
- AuditLog（配置变更与操作审计）
- ModerationQueue（可选：待处理内容队列）

## 7. 接口清单（草案）

### 7.1 登录与用户

- `/auth/login`：微信登录换取业务token
- `/user/profile`：用户信息
- `/user/bindPhone`：绑定手机号

### 7.2 智能体与对话

- `/agents`：智能体列表（含分类、排序、启用）
- `/agents/{id}`：智能体详情
- `/chat/send`：发送消息，返回回复 + 推荐卡片
- `/chat/history`：历史消息

### 7.3 课件与交易

- `/products`、`/products/{id}`：课件列表与详情
- `/orders/create`：创建订单
- `/pay/wx/prepay`：获取微信预支付信息
- `/orders/{id}/status`：订单状态
- `/purchases`：已购列表

### 7.4 论坛

- `/posts`、`/posts/{id}`：帖子列表与详情
- `/posts/create`：发帖
- `/comments/create`：评论
- `/likes/toggle`：点赞/取消
- `/reports/create`：举报

### 7.5 后台（管理端）

- `/admin/agents/*`：智能体配置管理（增删改查、发布、回滚）
- `/admin/moderation/*`：举报与处置
- `/admin/audit/*`：审计日志查询

## 8. MVP范围建议（第一版可上线）

- 必做：
  - 首页：对话 + 推荐卡片 + 购买与已购交付
  - 技能课超市：多智能体入口 + 后台可配置
  - 我的：订单、已购、基础设置
- 可选（后置）：
  - 论坛圈子/话题运营、搜索、精细化推荐
